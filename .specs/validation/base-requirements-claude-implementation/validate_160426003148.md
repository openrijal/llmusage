2. **Codex model name**: Stored as "codex-openai" — actual model (GPT-5) not exposed per-turn in logs

4. **Time zones**: SQLite datetime functions use UTC; recorded_at timestamps preserve original timezone from source
5. **First sync**: If LiteLLM fetch fails, falls back to hardcoded pricing for ~11 models only
